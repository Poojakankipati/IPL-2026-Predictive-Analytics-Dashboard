# IPL 2026 Predictive Analytics Dashboard

## Overview
This project is an AI & Machine Learning powered IPL analytics dashboard built using Power BI and predictive modeling techniques.

## Features
- Team Win Distribution Analysis
- IPL 2026 Win Probability Prediction
- Predicted Points Table
- Match Winner Prediction
- Feature Importance Analysis
- Win Probability Gauge
- Qualified vs Non-Qualified Teams

## Technologies Used
- Power BI
- Python
- Machine Learning
- XGBoost
- Pandas
- Data Visualization

## Machine Learning Insights
The dashboard uses historical IPL data (2008–2025) and predictive analytics models to estimate:
- Match outcomes
- Team rankings
- Championship probabilities
